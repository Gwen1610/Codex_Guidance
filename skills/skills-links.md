# Codex Skills 收藏链接

以下链接来自官方与社区中较受欢迎的技能集合与目录，适合用来快速发现、筛选并安装技能。

## 官方与权威目录

- [openai/skills](https://github.com/openai/skills)  
  官方 Skills Catalog，包含系统/精选/实验技能与安装说明。
- [Codex Skills 官方文档](https://developers.openai.com/codex/skills/)  
  说明技能结构、安装方式与使用位置。

- [Codex Skills Registry](https://skillregistry.dev/)  
  面向 Codex CLI 的技能目录，可直接检索与一行安装。

  Vercel 开源的技能合集，覆盖前端最佳实践、UI/UX 规范、部署等。
