# Codex Guidance

这是一个用于记录 Codex CLI 学习与配置理解的资料库，重点覆盖 `config.toml` 与 `skills` 两大模块。

## 目录结构

- `config/`：配置示例与配置参考解读
- `skills/`：技能相关资源与收藏链接

## 快速入口

- Sample Configuration（官方示例，已保存）：`config/config.sample.toml`
- 配置参考（全量字段清单）：`config/config-reference-guide.md`
- 配置参考中文解读：`config/config-reference-guide.zh.md`
- Skills 收藏链接：`skills/skills-links.md`

## 适用人群

- 刚开始使用 Codex CLI 的新手
- 需要理解或整理 `config.toml` 的用户
- 需要收集与管理 Skills 资源的使用者
